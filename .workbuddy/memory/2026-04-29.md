| 文件 | 改动 |
|------|------|
| `src/cli/main.ts` | 新增 --dashboard/-d 选项；新增 executeWithDashboard() 方法（惰性加载 ink + React + App）；run/committee 命令均支持 |

### 设计决策
- **惰性加载**: ink/React/App 通过 `import()` 动态导入，非 dashboard 模式零内存开销
- **事件桥接**: DashboardEventBridge 保留原始回调（先调用原回调，再 emit 事件），不破坏 Phase 1 行为
- **输出缓冲**: 最多 500 行，超限裁剪到 400 行，防止长时间运行内存增长
- **完成状态**: 任务结束后 Dashboard 显示最终结果，800ms 后 unmount 回到标准输出
